# Pomodoro Timer App

Flutter приложение для управления временем по методике Pomodoro с матрицей Эйзенхауэра.

## Функции
-  Таймер Pomodoro (25/5 минут)
-  Управление задачами через матрицу Эйзенхауэра
-  Календарь и статистика продуктивности
-  Многоязычная поддержка (русский/английский)
-  Система пользователей
-  Тёмная тема

## Установка
1. Убедитесь, что установлен Flutter SDK
2. Клонируйте проект
3. Выполните: `flutter pub get`
4. Запустите: `flutter run`

## Зависимости
- Flutter SDK
- Стандартные пакеты Flutter
